# Dự Án: Website Học Tập Ngôn Ngữ

## ✨ Ý Tưởng

Tạo ra một trang web học tập cho phép người dùng có thể học ngôn ngữ một cách dễ dàng hơn, trực quan và thân thiện. Mục tiêu là giúp người dùng rèn luyện kỹ năng ngôn ngữ qua các bài học tương tác, trò chơi từ vựng, và hệ thống theo dõi tiến độ.

## 🛠️ Công Nghệ Sử Dụng Dự Kiến

- **HTML**
- **CSS**
- **Java**

## 🚀 Mục Tiêu Chính

- Giao diện đơn giản, thân thiện với người học.
- Hệ thống bài học phân cấp theo trình độ.
- Tích hợp trò chơi hoặc bài kiểm tra giúp ghi nhớ từ vựng tốt hơn.
- Theo dõi tiến độ học tập và nhắc nhở hàng ngày.
